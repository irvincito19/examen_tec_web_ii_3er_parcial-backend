# Backend - Examen Práctico - TecWeb II

Este repositorio contiene la lógica de servidor y los servicios API desarrollados para la solución del examen.

## Descripción General
Se encarga de la gestión de datos, reglas de negocio y la exposición de endpoints para ser consumidos por el cliente (frontend).

### Tecnologías utilizadas
* **Java / Spring Boot**
* **Maven** (Gestión de dependencias)
* **H2** Para persistencia
